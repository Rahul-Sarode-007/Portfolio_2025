SSAS

Ship Security Alert System

Overview

The Ship Security Alert System (SSAS) is a mandatory security system required under the International Ship and Port Facility Security (ISPS) Code. It enables vessels to transmit covert security alerts to designated authorities when the ship is under threat of piracy, terrorism, or other security incidents, without alerting potential attackers onboard.

Technical Features

  • Covert Alert Transmission: Sends security alerts without any visible or audible indication onboard
  • Dual Transmission Paths: Utilizes both satellite and terrestrial communication for redundancy
  • Multiple Activation Points: Strategically located activation buttons on bridge and other secure locations
  • Continuous Monitoring: 24/7 connection to designated competent authorities
  • Tamper Detection: Protected against unauthorized access and manipulation
  • Position Reporting: Automatically includes vessel identity and position in alert message

Alert Information Transmitted

Security Alert Data

  • Ship identity (IMO number, call sign, name)
  • Current position (latitude/longitude)
  • Date and time of alert activation
  • Alert type (if multiple activation points)
  • Continuous position updates (if configured)

Benefits

  • Enhanced Security: Enables rapid response to security threats including piracy and terrorism
  • Crew Safety: Allows crew to alert authorities without endangering themselves
  • ISPS Compliance: Meets mandatory international maritime security requirements
  • Global Coverage: Satellite transmission ensures alerts reach authorities anywhere in the world
  • Immediate Response: Enables coordinated response from naval forces and coast guards
