AIS

Automatic Identification System

Overview

The Automatic Identification System (AIS) is an automated tracking system that uses VHF radio frequencies to exchange vessel information between ships and shore stations. AIS enhances maritime safety by providing real-time vessel position, course, speed, and other critical navigation data, enabling collision avoidance and improving situational awareness for mariners and vessel traffic services (VTS).

AIS Data Types

Dynamic Information (Updated Frequently)
  • Vessel position (latitude/longitude)
  • Position accuracy and timestamp
  • Course over ground (COG)
  • Speed over ground (SOG)
  • Heading
  • Rate of turn (ROT)
  • Navigational status (underway, at anchor, moored, etc.)
Static Information (Updated Every 6 Minutes)
  • IMO number (Class A only)
  • Call sign and vessel name
  • Vessel type and dimensions
  • Type of position fixing device
  • Location of position reference point
Voyage-Related Information (As Required)
  • Destination
  • Estimated time of arrival (ETA)
  • Draught
  • Cargo type (dangerous goods)
  • Number of persons onboard

Technical Features

  • Autonomous Operation: Continuous automatic transmission without operator intervention
  • VHF Channels: Operates on dedicated maritime VHF channels (161.975 MHz and 162.025 MHz)
  • SOTDMA Protocol: Self-Organizing Time Division Multiple Access for efficient channel usage
  • Integration Capability: Interfaces with ECDIS, radar, and other navigation systems
  • Collision Avoidance: Provides CPA (Closest Point of Approach) and TCPA (Time to CPA) calculations
  • VTS Support: Enables Vessel Traffic Services to monitor and manage vessel movements

Regulatory Requirements

SOLAS Chapter V - Regulation 19

  • All Ships ≥300 GT: Engaged on international voyages
  • All Cargo Ships ≥500 GT: Not engaged on international voyages
  • All Passenger Ships: Irrespective of size
  • Implementation Date: Phased implementation completed by July 1, 2008
  • Class A Transponder: Required for SOLAS vessels
  • Performance Standards: IMO Resolution MSC.74(69) and ITU-R M.1371
  • Annual Testing: Verification of proper operation required

Applications & Benefits

  • Collision Avoidance: Real-time awareness of surrounding vessel traffic
  • Port Operations: Improved traffic management and berth planning
  • Search and Rescue: Rapid identification and location of vessels in distress
  • Fleet Management: Real-time vessel tracking and route optimization
  • Maritime Security: Vessel identification and monitoring for security purposes
  • Environmental Protection: Monitoring vessel movements in sensitive areas
  • Regulatory Compliance: Meets SOLAS Chapter V requirements
